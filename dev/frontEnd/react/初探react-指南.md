# 初探react-指南
2018/11/6

### install nodejs on mac

访问：
https://nodejs.org/zh-cn/

下载稳定版本到 mac 下安装即可。

为了加速安装过程，设置仓库
```bash
npm config set registry https://registry.npm.taobao.org --global

```


### React 的项目初始化工具
```bash
$ sudo npm install -g create-react-app
$ npx create-react-app my-app
$ cd my-app
$ npm start

```



### 构建生产版本
```bash
$ npm run build

```
